## User Stories
### Epic 1: User Management
**As a user, I want to:**
- Register with security questions
- Reset my password if forgotten
- View and update my profile

### Epic 2: CRUD Operations
**As an admin, I want to:**
- Add new artists and albums
- Update existing music records
- Delete records via front-end interface
